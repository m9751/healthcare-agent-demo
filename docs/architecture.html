MCP Connector — Open Standard

Any Mule app becomes an MCP server with the MCP Connector v1.3.1 (on Exchange). This means any AI agent — Claude, GPT, Agentforce, or custom — can discover and call MuleSoft APIs using the open Model Context Protocol. No proprietary integration required.
